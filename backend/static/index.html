🔑 获取 API Key

注册 云雾API，在控制台创建 API Key。

推荐使用 image2 模型，支持 2K、4K 图片生成。

💰 价格说明

平均每张约 0.5 元 左右。系统会自动分组生成，不同分组价格略有差异，由中转站决定。

📝 使用说明

  • 填入 Base URL 和 API Key，点击保存
  • 切换到「图片生成」标签，选择比例和模型
  • 输入提示词（支持中文/英文），点击生成
  • 支持独立提示词模式：每行一个提示词，各生成一张
  • 「风格模板」可快速套用结构化提示词
  • 生成结果可下载、复制链接、重试

🎯 特色功能

  • 实时监控：查看每次请求的完整 JSON
  • 生成历史：保存所有生成记录
  • 案例展示：内置 453 个案例参考
  • 元数据编辑：批量修改图片 EXIF 信息
